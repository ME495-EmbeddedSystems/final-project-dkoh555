# Speak
This package speak out the translated text
## Setup Task
Install the `google_speech` api via
```
pip install google-speech
```
Install the `pyttsx3` via
```
pip install pyttsx3
```
Install the `SoX` via
```
sudo apt-get install sox libsox-fmt-mp3
```
