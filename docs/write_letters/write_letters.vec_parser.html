write_letters.vec_parser module

Parser node that parses the all waypoints of a set of letters to a path.

Parameters

offset_letter: The gap between the letters offset_standoff: The offset between pen and board when standing off offset_penup: The offset for pen lifting up scaling: The scaling factor for how big each character is x_start: The start X coordinate z_start: The start Z coordinate
